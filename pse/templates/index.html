PHStock.xyz

A simple JSON API for PSE

Usage

  • Get All Stocks
    • http://phstock.xyz/stocks
  • Most Active Stocks
    • http://phstock.xyz/stocks/active
  • Top Gainers Stocks
    • http://phstock.xyz/stocks/winners
  • Top Losers Stocks
    • http://phstock.xyz/stocks/losers
  • Get Specific Stock
    • http://phstock.xyz/stock/${security_symbol}

Examples
  http://phstock.xyz/stock/ALI
  http://phstock.xyz/stock/UBP
  http://phstock.xyz/stock/NOW
